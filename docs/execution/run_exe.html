Running an O'Caml executable

Once you have created an executable (see Creating executables), you can execute it by right clicking on it in the navigator view (it should appear with the icon if it is a byte-code executable, or with the icon if it is a native executable), and selecting Run As > Ocaml Executable.

Actually, this is a launch shortcut that automatically creates a launch configuration for you, and runs it. But you can also create a launch configuration manually by clicking on Run > Open Run Dialog... in the main menu.

You also use this dialog to configure the command-line parameters (see Configuring command-line parameters).

Module browser

The O'Caml plug-in includes an O'Caml library module browser, which is available in the default layout of the O'Caml perspective (see Opening the O'Caml perspective) as a fast view. To open it, click on the icon in the lower-left corner of the Eclipse window. You can also open it by clicking on Window > Show View > OCaml Browser.

The module browser is divided in two parts: the left pane shows a hierarchical view of the modules and module definitions, and the right pane displays a description of the element selected in the left pane.

Indentation

The O'Caml editor indents code automatically as you type it, by using a set of simple rules based on the O'Caml syntax. For example, when you type Enter the end of the line is analyzed to determine if the next line must be indented.

You can also use the Tab key at the beginning of a line to indent it, and the Shift + Tab key combination to dedent.

If you want to indent or dedent several lines at once, select them first and then use Tab or Shift + Tab.

Note: the Tab key is also used for inserting the "->" characters when it is used in the middle of a line, to insert the "[]" characters after "::" or to complete "fn" into "function".

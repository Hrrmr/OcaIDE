Completion

Code completion is available in the O'Caml modules and interfaces editor.

To activate it, type Ctrl + Space (by default). You will get a list of completion propositions available in the current context.

For example, if you type an uppercase "B" in an empty module file and then hit Ctrl + Space, you will get the following completion box, showing all the modules starting with a "B":

You can then continue to type to further narrow down the list of propositions. You can also select a proposition by using the Up, Down, Page Up, Page Down, or by clicking on it.

Each time you select a proposition, the help pop-up on the right changes to reflect your selection, by displaying useful informations on the selected element extracted from the interface (mli) file in which this element is located.

Once the element you were looking for is selected, type Enter to insert it into the editor at the current position. You can also double-click on the element to insert it.

In our example, this would insert the word "Buffer" into the editor.

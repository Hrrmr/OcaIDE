Hyperlinks

The editor for module files and ocamlyacc files allows you to jump to the definition of an element by clicking on it while pressing the Ctrl key (Command key on Mac).

Click on a variable, function, type constructor, etc. to jump to its definition.

Click on the name of a module in an "open" directive to open the corresponding interface in a new editor.

Note: for hyperlinks between modules to work, the project paths must be configured (see Configuring project paths).

Note: Start by pressing Ctrl (or Command) and hover over the element. The link can take some time (several seconds) to appear the first time when hyperlinks are not yet initialized.

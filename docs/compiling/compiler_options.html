Compiler Options

This page explains how to set the compiler options on OCaml Managed Projects. For OCaml Makefile Projects, see Configuring Makefile targets.
For Ocamlbuild projects, configure the ocamlbuild targets, the libraries, the compiler flags and the linker flags in the project properties page, under the section named Project.

To configure the compiler options that must be applied to a module, right-click on the module in the navigator view, and select properties. Then, select the OCaml Module category:

In this properties page, you can add compilation flags by typing a flag in the text field and clicking Add. You can also remove, change and reorder flags by clicking on the corresponding buttons.

You must enter the flags as you would on the command line, but without spaces. For example, the "-w a" flag (to disable all warnings) must be split in two: "-w" and "a".

You can also add some global compiler flags that will be applied to each module in the project, by opening the project properties page (right-click on a project in the navigator, then select properties), and selecting the OCaml Build Flags section:

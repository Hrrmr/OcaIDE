Creating a new project

To create a new OCaml project, right click (or Ctrl + Click on Mac) on empty space in the navigator view. Alternatively, you can also click on File > New.

If you are in the OCaml perspective (see Opening the OCaml perspective), you should see the following menu:

From this menu, you can choose between four different kinds of OCaml projects:

  • OCaml Project (ocamlbuild): The building of your projects is managed by Ocamlbuild (for Ocaml versions >= 3.10 only) .
  • OCaml Managed Project: The building of your projects is managed by OcaIDE. Use it only for simple projects.
  • OCaml Makefile Project: The building of your projects is managed by a Makefile. A generic Makefile is provided, so that you only have to specify some rules for your project to compile correctly.
  • OCaml Empty Makefile Project: Use it when you want to import a project, and you don't need the plug-in to generate a generic Makefile.

Once you have clicked on one of the three project options, you get the following dialog box:

Type the name you want to give to your project (avoid spaces and special characters), and click Finish.

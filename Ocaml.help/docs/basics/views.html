Opening Views

When you open the OCaml perspective for the first time, all the most useful views are automatically opened for you. That includes the following views:

  • Problems
  • Ocaml Toplevel
  • Ocaml Compiler Output
  • Console
  • Error Log
But you can choose to open other views or close some views that are not useful to you.

If you want to open or reopen a view, click on Window > Show View and select the view you want to open. If the view you're looking for is not available in this menu, click on Other... to open a window which shows all the views available in Eclipse.

You can also reset the OCaml perspective to its default layout at any moment by clicking on Window > Reset Perspective in the main menu while the OCaml perspective is open.

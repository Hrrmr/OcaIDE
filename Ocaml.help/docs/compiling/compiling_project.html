Compiling an OCaml project

There are several ways to compile an OCaml project depending on the project type and on the options you set:

  • If Project > Build Automatically is checked, saving a modified file will trigger a recompilation of the project. So, to force a file to recompile, you can modify it (adding and then removing a space for example) and save it.
  • If Project > Build Automatically is not checked, you can compile a project by selecting the project in the navigator view and then clicking on Project > Build Project, or by right-clicking on the project in the navigator view and clicking Build Project in the context menu. The project will only be rebuilt if one of its files was modified.
  • You can force a full recompilation of a project by cleaning it (see Cleaning a project).
  • For an OCaml Makefile Project or an OCaml Empty Makefile Project, right-click on the project in the navigator view, and select Make Project (the Makefile targets must be correctly configured, see Configuring Makefile targets)

Debugging under Windows

To make the debugger work under Windows, follow these steps:

  • Install Cygwin (install the devel packages, but don't install the OCaml version from Cygwin)
  • Download OCaml sources
  • Compile OCaml with Cygwin, and install it
  • Set environment variables:
    • Right-click My Computer, and then click Properties
    • Click the Advanced tab
    • Click Environment variables
    • Remove any references to any other installation of OCaml from your path
    • Add the folder you installed OCaml in to your system path
      (for example, "c:\cygwin\usr\local\bin")
    • Set the OCAMLLIB variable in your system variables
      (for example, OCAMLLIB = "c:\cygwin\usr\local\lib\ocaml")

Then, start the debugger as usual.

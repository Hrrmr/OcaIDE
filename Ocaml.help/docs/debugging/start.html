Starting the debugger

To start debugging a byte-code executable (the OCaml debugger doesn't work on native executables), right-click on it in the navigator view, and select Debug As > Ocaml Executable:

This will open the OCaml Debug perspective, which provides a view layout designed for debugging OCaml executables.

When you start the debugger, the program is loaded into memory, but the execution of the program has not started yet. To start it, see Stepping.

Note: Do not switch manually to the OCaml Debug perspective, as this would not start the debugger.

Type informations

The OCaml compiler provides the "-dtypes" command line option to make it display the types it inferred for all the elements in the files it was asked to compile.

The OCaml plug-in uses this functionality to show the types in pop-ups when you hover the mouse over an element in the editor:

For it to work, the current file must be compiled with the "-dtypes" option, and mustn't have been modified since the last time it was compiled.

The "-dtypes" option is automatically put on OCaml Managed Projects, and is defined in the OCaml Makefile Projects, but you have to set it manually if you are using an OCaml Empty Makefile Project.

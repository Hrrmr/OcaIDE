Help on expected parameters

Right after you enter an element in the editor using completion (see Completion), an information pop-up appears to show the parameters expected by the function, constructor or exception you just entered.

This pop-up is automatically hidden when it is no longer necessary (when the cursor in the editor gets out of the lexical scope in which it is defined).
You can also close it manually by hitting the Escape key.

Note: This feature uses the interface definition of the element. So, for it to work, the element must be defined in an interface accessible from the paths you defined (see Configuring paths and Configuring project paths).

Code formatter

The OCaml plug-in includes a code formatter for module files. You trigger it by clicking on Ocaml > Format or by pressing the F3 key (configurable in the key preferences).

If several lines are selected in the editor, only those lines will be reformatted. Otherwise, the whole contents of the editor will be reformatted.

This formatter does the following:

  • Indentation based on nesting level
  • Spacing between words (variables, separators, etc.)
  • Wrapping of long comments
  • Merging of consecutive small comments
  • Deleting of redundant blank lines

If you want to protect a comment from the formatter (for example if you have made some ASCII-art), start your comment with (*| instead of (*. Ocamldoc comments are never modified by the formatter.

Note: Make the difference between this formatter and the automatic indentation provided by the editor (see Indentation). The automatic indentation happens as you enter code for the first time, whereas the code formatter is called on existing code.
